# Unity A* Pathfinding Prototype

This project is a grid-based A* pathfinding implementation built in Unity.

It also includes a singular level with an AI game object that can travel from one point of the map to the other using an A* implementation in conjunction with a state machine. 

## Features

- Grid-based node graph
- Automatic node graph generation at runtime
- A* pathfinding algorithm implementation
- Neighbour detection system
- Path reconstruction from target node to start node
- AI state machine for game objects

## Technologies

- C#
- Unity Engine

## Purpose of the Project

This project was created as an experiment in implementing algorithms commonly used in games and simulations. It mainly focuses on the underlying logic of pathfinding rather than on building a full game.

Future work may involve integrating the system into a larger gameplay project.
